**Суть проекта**: Этот проект генерирует случайные рецепты пиццы с разными видами теста, соусов и начинок.

**Примеры использования**: 
Рецепт случайной пиццы:
Основа: тонкое тесто
Соус: томатный соус
Начинка: пепперони, грибы, сыр моцарелла.
